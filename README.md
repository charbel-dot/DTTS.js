# 🤖 [Discord.js] bot Tutorial

## 🏴 Dependencies

`Discord.js` **REQUIRED**

`Node.js v14+` **REQUIRED**

## ✔ Installation

This will install all **node packages** automatically

```bash
npm i -g discord.js
```
